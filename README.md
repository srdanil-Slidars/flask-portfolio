# Flask Portfolio

Стартовый каркас портфолио-сайта на Flask.

## Запуск проекта

1. Создать виртуальное окружение
2. Установить зависимости:
   pip install -r requirements.txt
3. Запустить:
   python run.py

## Структура
- app/ — приложение
- templates/ — HTML-шаблоны
- static/ — стили, скрипты, изображения
